Курс по продуктивности с использованием ИИ.

## Структура проекта

- `10 structure/` — структура курса
- `20 transcripts/` — транскрипты встреч
- `40 feedback/` — обратная связь

## Генерация сайтов

В проекте два **один** сайт, которые публикуются в **разные репозитории**:

1. **Лендинг курса** — главная страница (из `website_output/`)

### Генерация лендинга для GitHub Pages

```bash
python3 generate_landing_for_pages.py
```

Это создаст папку `docs/` с `index.html` (последняя версия лендинга).

### Генерация базы знаний для GitHub Pages

```bash
cd "30 aimindset"
python3 generate_knowledge_base_for_pages.py
```

Это создаст папку `docs/` в корне проекта с базой знаний.

**См. `SETUP.md` для подробных инструкций по настройке каждого репозитория.**
